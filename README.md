## Getting Started

To get a local copy up and running follow these simple example steps.

### Setup

```bash
https://github.com/elasfarc/country-info.git
cd ./math-magicians
```

### Install

```bash
npm install
```

### To Run it

```bash
npm start
```

### Build

```bash
npm run build
```
